10. QUICK REFERENCE
================================================================================

To Use Theme Colors in Components:

  Method 1 (Tailwind):
    <div className="bg-brand-primary text-brand-accent">...</div>

  Method 2 (CSS Variables):
    <div style={{ background: 'var(--brand-primary)' }}>...</div>

  Method 3 (Hook):
    import { useBrandTheme } from '../hooks/useBrandTheme';
    useBrandTheme('roland');

To Scale to Multiple Brands:

  1. Scrape brand data:
     python3 orchestrate_brand.py --brand yamaha --max-products 50

  2. Run forge pipeline:
     python3 forge_backbone.py

  3. Test in browser:
     Select products from different brands → Colors change!

To Add New Brand Colors:

  1. Update frontend/src/styles/brandThemes.ts
  2. Update backend/forge_backbone.py BRAND_THEMES
  3. Scrape brand data
  4. Run forge_backbone.py
